# 📊 Employee & Project Management Dashboard

## 🚀 Project Overview
This project analyzes employee data and project status using SQL and Power BI. It provides insights into project allocation, department performance, and employee contributions.

---

## 🛠️ Tools & Technologies
- Microsoft SQL Server
- Power BI
- SQL (Joins, CTEs, Data Transformation)

---

## 📂 Project Files
- `Project_Overview.pbix` → Interactive Power BI Dashboard
- `SQLQuery1.sql` → Data extraction and transformation queries

---

## 📊 Key Features
- Employee performance tracking
- Department-wise analysis
- Project status (Upcoming vs Completed)
- Salary insights and role distribution

---

## 🧠 SQL Highlights
- Used Common Table Expression (CTE) for project status
- Joined multiple tables:
  - Employees
  - Departments
  - Project Assignments
- Created unified project tracking system
